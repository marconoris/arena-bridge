# Are.na Bridge 1.0.1-beta.3

This beta release adds an explicit mobile fallback for folder-level sync actions, so the core Are.na flows remain reachable even when mobile file context menus do not expose plugin entries consistently.

## Highlights

- New commands let you sync or refresh the current note folder from the Command Palette.
- Mobile users can trigger folder/channel actions without depending on long-press file menus.
- Existing desktop context menu actions remain unchanged.

## Notes

- This release should still be published on GitHub as a `Pre-release`.
- Mobile support remains beta and should be validated on a real device.
- BRAT users should confirm the installed version shows `1.0.1-beta.3`.
