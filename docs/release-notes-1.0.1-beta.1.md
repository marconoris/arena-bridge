# Are.na Bridge 1.0.1-beta.1

This is a public beta release focused on validating mobile support while keeping the existing desktop flows intact.

## Highlights

- Mobile support is now available as a beta.
- Channel and confirmation modals have been hardened for smaller screens.
- Channel selection layout now degrades more cleanly on narrow viewports.
- Confirmation before loading additional channel pages now uses an in-plugin modal instead of the browser dialog.

## Notes

- This release should be published on GitHub as a `Pre-release`.
- Mobile support is still being validated and may have rough edges.
- Desktop remains the reference environment during this beta phase.

## Recommended test flows

- Browse my channels
- Get blocks from a channel with paginated loading
- Push note to Are.na
- Create channel
- Open block in Are.na
